#### 安装教程

1.  首先安装相关依赖

```
sudo apt install deepin-wine6-stable spark-dwine-helper | store.spark-app.spark-dwine-helper fonts-wqy-microhei fonts-wqy-zenhei
```

2.  使用dpkg进行安装

```
sudo dpkg -i spark-deepin-wine-seetong_1.0.4.4_all.deb
```
